# DataScience_ALURA

Projeto desenvolvido durante o curso **Data Science: Explorando e Analisando Dados**, da Alura. O repositório reúne notebooks de análise exploratória de dados com foco em estatística descritiva, tratamento inicial de dados e visualização com Python.

## Sobre o projeto

O objetivo foi praticar a leitura, limpeza e exploração de bases reais para responder perguntas de negócio e de análise de dados. Ao longo dos notebooks, são trabalhados diferentes cenários:

- avaliação de filmes com base no dataset `ratings.csv`;
- exploração do catálogo de filmes do dataset `movies.csv`;
- análise de filmes e atributos do TMDB com `tmdb_5000_movies.csv`;
- análise de vendas da cafeteria Serenatto com `serenatto_2sem_2023.csv`.

## O que foi explorado

- leitura de arquivos CSV com `pandas`;
- inspeção de estrutura, tipos de dados e valores ausentes;
- criação de métricas como média, mediana, distribuição e contagem;
- agrupamentos e filtros com `query`, `groupby` e `value_counts`;
- visualizações com `matplotlib` e `seaborn`;
- comparação entre filmes, notas e padrões de vendas.

## Estrutura do repositório

- `DataScience_COMPLETO.ipynb`: versão consolidada do estudo.
- `Base_Dados/`: pasta com dados de apoio.
- `serenatto_2sem_2023.csv`: base de vendas usada em uma das análises.
- `tmdb_5000_movies.csv`: base de filmes do TMDB.
- `.\Certificado`: certificado de conclusão do curso.

## Tecnologias utilizadas

- Python
- Jupyter Notebook
- pandas
- numpy
- matplotlib
- seaborn

## Como executar

1. Clone este repositório.
2. Abra um dos notebooks no VS Code ou no Jupyter.
3. Instale as dependências, se necessário:

```bash
pip install pandas numpy matplotlib seaborn
```

4. Execute as células na ordem em que aparecem no notebook.

## Resultado

Ao final, o projeto mostra como transformar bases brutas em análises visuais e interpretáveis, destacando padrões de notas de filmes, características do catálogo e comportamento de vendas.
